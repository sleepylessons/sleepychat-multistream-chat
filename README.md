# Multi-Stream Chat - SleepyChat
Mutistream chat app for Twitch, Tiktok, Youtube, and Kick.

[https://sleepychat.org/](https://sleepychat.org)

## About
This is the issue tracker for [the multi-stream chat app SleepyChat](https://sleepychat.org)

SleepyChat enables you to combine your Twitch, TikTok, YouTube, and Kick chats into one cohesive feed.

## Submitting Issues
If you an encounter an issue with your multi-stream chat, please [create an issue](https://github.com/sleepylessons/sleepychat-multistream-chat/issues/new)

## Feature Requests

If you want to request a Feature, you can also [create an issue](https://github.com/sleepylessons/sleepychat-multistream-chat/issues/new)
